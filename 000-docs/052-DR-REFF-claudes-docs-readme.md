# claude doc workspace

This folder captures working notes produced while collaborating with claude and other ai agents. Each subdirectory focuses on a specific artifact type so historical context is easy to recover.

## structure

| folder | use |
| --- | --- |
| `analysis/` | deep dives, audits, and root-cause investigations. |
| `implementation/` | step-by-step fixes, patch files, and deployment guides. |
| `tasks/` | taskwarrior exports, worklogs, and backlog snapshots. |

Add new notes inside the appropriate folder and include a short preamble (date, author, purpose) so future sessions can pick up context quickly.
