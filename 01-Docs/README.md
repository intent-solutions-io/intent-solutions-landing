# 📚 Intent Solutions Landing Page Documentation

## Project Overview
Landing page and web presence for Intent Solutions services.

## Directory Structure

```
docs/
├── PRDs/           # Product Requirement Documents
│   ├── template.md
│   ├── create-prd.md
│   ├── generate-tasks.md
│   └── process-task-list.md
├── ADRs/           # Architecture Decision Records
├── specifications/ # Technical specifications
└── tasks/          # Task tracking and planning
```

## Development Workflow

### 1. Planning Phase
- Create PRD using `PRDs/create-prd.md`
- Generate tasks with `PRDs/generate-tasks.md`
- Save to appropriate directories

### 2. Implementation Phase
- Use `PRDs/process-task-list.md` to work through tasks
- Update task status as you progress
- Document decisions in ADRs
