```bash

open 01-dark-professional.html
```

## option 1: dark professional

**file**: `01-dark-professional.html`

**colors**:
- background: #0a0a0a (near black)
- text: #ffffff (white)
- accent: #8b5cf6 (purple)
- gradient: purple glow effects

**vibe**: premium, sophisticated, enterprise
**like**: vercel, linear, intelligent ai studios
**best for**: showing serious technical capabilities
